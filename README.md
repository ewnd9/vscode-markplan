# `markplan`

## Usage

```sh
$ yarn vsce package
$ code --install-extension markplan-*.vsix
```

## Development

```sh
$ yarn build:watch &
$ code --disable-extensions --extensionDevelopmentPath=$PWD $PWD # reload to test changes
```
